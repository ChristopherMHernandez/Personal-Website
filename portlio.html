When we got the color of marble it had told the machine where the supplies was going to in the project. We had to use color sensors to read the barcode to tell the program what it was going to do. Traffic Signal
In project I had built a traffic light using a finite state machine from what i had learned in the previous lab. I had written the code in Verilog. In order to make the traffic light I had used an ZYBO board. I had program the board using the code that I had program in verilog and uplloaded it to the board. I was creating a traffic light at intersection with a farm road and busy roadway.
Eletronic Security Signal
In this project, I had used my AD2 to create an electronic alarm. I had used a combination of resistors and opamps to make the the signal. When their was something obstructing our emitter a red light would turn on. If not a green light would stay mentioning that there was nothing blocking the emitter.
